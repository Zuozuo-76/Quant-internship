# Quant-internship

Static K-line dashboard and CSV outputs for the quant internship assignment.

## Contents

- `index.html`: public K-line dashboard
- `kline_data.js`: chart data used by the dashboard
- `data/daily_kline.csv`: daily OHLCV table with active buy/sell fields
- `data/minute_kline_20250401.csv`: minute-level sample table
- `data/factor_values.csv`: factor construction results
- `data/factor_formulas_zh.csv`: factor formulas
- `data/output_structure.csv`: output structure for each generated field table
- `assignment_outputs_zh.md`: Chinese explanation of fields, filling rules, and output layout

## GitHub Pages

To make this public as a website:

1. Open repository Settings.
2. Go to Pages.
3. Set Source to `Deploy from a branch`.
4. Select branch `main` and folder `/root`.
5. Save and wait for GitHub Pages to publish.
